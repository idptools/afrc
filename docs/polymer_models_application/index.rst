Polymer Models (Application)
=========================================================

This section mirrors :doc:`Polymer Models (Theory) <../polymer_models/index>`, but from the
practical side. For each model it provides (1) a few quick-start examples showing how to build
the object and pull out the quantities you are likely to need, and (2) the full autogenerated
code reference (every public method, its arguments, and return types).

If you are after the maths, parameters, and references for a model, see the theory section; if
you just want to use it, you are in the right place.
